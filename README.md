# homebrew-mac-dir-stat

Homebrew tap for [mac-dir-stat](https://github.com/Chartres/mac-dir-stat).

## Install

```sh
brew install --cask chartres/mac-dir-stat/mac-dir-stat
```

Or tap once and install short-form:

```sh
brew tap chartres/mac-dir-stat
brew install --cask mac-dir-stat
```

## Update

```sh
brew upgrade --cask mac-dir-stat
```

## Uninstall

```sh
brew uninstall --cask mac-dir-stat       # remove the app
brew uninstall --cask --zap mac-dir-stat # also remove preferences and saved state
```
